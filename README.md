I optimize my daily routine using **reinforcement learning** (Q-learning):

```python
# take a step
S = "busy" # state
A = "play" in { "play", "work" } # action
S' = "boom" # next state
R = -100 # penalty

# learn from an experience
Q[S,A] += alpha * (R + gamma * max(Q[S',:]) - Q[S,A])

# but, still I don't want to work...
A' = "play" # next action
```
